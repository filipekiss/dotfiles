# .dotfiles

> This is a work in progress. You can check my legacy configuration
> [here][legacy-dotfiles].

#### Why?

I've been versioning my dotfiles [for a while][dotfiles-first-commit] and I've
added a lot of configuration/utilities during all this time but I also have
a lot of leftover configuration files for software I don't even have installed
on my machine anymore.

I've tried [rebuilding] my configuration and also tried moving them to their
[own][repository] before, but this will still keep the leftover files
I mentioned before. So I decided to delete my configuration from my machine and
start over.
